# ⚡ Haxball Abilities

Special abilities for our Haxball futsal room! Pick a character, use abilities with your keyboard, and dominate the field.

---

## 🔥 How to Install

### Firefox (recommended)

1. Click this link: **[Download for Firefox](https://github.com/Dzithi/haxball-abilities/releases/latest/download/haxball-abilities.xpi)**
2. Firefox will ask "Add Haxball Abilities?" → Click **Add**
3. ✅ That's it! You're ready to play

### Chrome / Edge / Brave

1. Click this link: **[Download for Chrome](https://github.com/Dzithi/haxball-abilities/releases/latest/download/haxball-abilities-chrome.zip)**
2. Unzip the downloaded file (right-click → Extract All)
3. Open your browser and go to `chrome://extensions/`
4. Turn on **Developer mode** (the toggle in the top right corner)
5. Click **Load unpacked**
6. Select the folder you just unzipped
7. ✅ Done! Ignore the popup that appears when you restart your browser — just click X

## 🎮 How to Play

### Pick Your Character

Type `!char` in the room chat to see the available characters:

| Character | What they do |
|-----------|-------------|
| 🧨 **Bomber** | Place mines that freeze enemies, detonate them all at once, drop smoke to block the ball |
| ⚡ **Speedster** | Dash forward instantly, sprint super fast, become untouchable |
| 🛡️ **Guardian** | Place walls to block the ball and players, push enemies away, build barricades |
| 🧊 **Frost** | Create ice that slows enemies, freeze the closest enemy, slow down everyone |
| 🔥 **Striker** | Supercharge your next shot, pull the ball to you, rocket the ball at the goal |
| 👻 **Trickster** | Swap places with a teammate, teleport randomly, send an enemy back to their goal |

To pick one, type: `!char bomber` or `!char speedster` (or any name from the list)

### Use Your Abilities

By default, your ability keys are **1**, **2**, and **3** on your keyboard:

- **1** = Ability 1 (short cooldown, use often)
- **2** = Ability 2 (medium cooldown)
- **3** = Ability 3 (long cooldown, big impact)

Just press the key while playing! You need to be on the field (not spectating).

### Change Your Keys

Click the ⚡ icon in your browser toolbar → change the keys to whatever you prefer.

## ❓ FAQ

**Q: Nothing happens when I press 1/2/3?**
- Make sure you clicked inside the game first (the game needs to be focused)
- Make sure the extension is installed (you should see the ⚡ icon in your toolbar)
- Abilities only work when you're on the field during a game

**Q: Can I change character mid-game?**
- Yes! Type `!char <name>` anytime

**Q: What's my current character?**
- Type `!char` with no name to see it
